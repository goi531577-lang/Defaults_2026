Дефолты по облигациям — 2026

Одностраничный сайт для ручного учета дефолтов по выпускам облигаций. Подходит для GitHub Pages, работает без сервера, сохраняет данные в браузере, позволяет отдельно учитывать технический и реальный дефолт, увеличивать сумму задолженности и добавлять новые выпуски к уже существующему эмитенту.

Ручной ввод GitHub Pages Разделение: технический / реальный Экспорт и импорт JSON

Добавить дефолт

Запись можно создать с нуля или отредактировать уже существующую. Один эмитент может иметь несколько выпусков.

Инструменты

Данные хранятся локально в браузере. Для переноса на другой компьютер можно выгрузить JSON и затем импортировать его.

Для публикации на GitHub Pages достаточно загрузить этот HTML-файл как index.html в репозиторий и включить Pages в настройках репозитория.

Список эмитентов и выпусков

По каждому выпуску можно увеличить задолженность, изменить тип дефолта, отредактировать запись или удалить ее.

Увеличить задолженность

Добавьте сумму, на которую нужно увеличить задолженность по выбранному выпуску.
